Installation

Lire le fichier lisezmoi.txt de l'archive pour les infos de dernière minute.

Pour installer le programme il suffit de dézipper l'archive dans le répertoire souhaité (par exemple /usr/local/corewarrior ou c:\corewarrior).

Pour exécuter les programmes en mode graphique, lancer les scripts "editer" ou "editer.bat" (pour l'éditeur) et "moteur" ou "moteur.bat" pour le moteur.

Il faut éditer ces scripts pour changer le répertoire du programme (la variable CORE_HOME, y placer le répertoire de votre programme). Si vous utilisez comme machine virtuelle le JRE (Java Runtime Environment), remplacer : 

java -classpath $CLASSPATH:bin.jar:lib.jar casa.corewarrior.editeur.Editeur $*

par

jre -cp bin.jar:lib.jar casa.corewarrior.editeur.Editeur $*

Ou, pour les utilisateur Windows : 

java -classpath %CLASSPATH%;bin.jar;lib.jar casa.corewarrior.editeur.Editeur %1

par

jre -cp bin.jar;lib.jar casa.corewarrior.editeur.Editeur %1

Si vous rencontrez des problèmes pour installer ce programme, n'hésitez pas à m'envoyer un mail après avoir lu cette doc et celle de votre VM.

Merci de me contacter si vous dénichez des bugs. Pour ce faire, il m'est indispensable de disposer du message d'erreur (du type "java.lang.NullPointerException..." par exemple) accompagné d'un bref descriptif des circonstances où survient le problème. Pour intercepter le message d'erreur dans un fichier, dans le script, après java ..., sur la meme ligne : >erreurs.txt. Les messages d'erreur se trouveront dans le fichier erreurs.txt du répertoire du programme (ou mieux, pour les utilisateurs UNIX, >~/erreurs.txt).
