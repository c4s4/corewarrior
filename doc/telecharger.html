A installer avant Core Warrior

Ce programme nécéssite une machine virtuelle (VM) java 1.1 pour fonctionner. Sun met gratuitement à disposition des utilisateurs une VM pour les PC et stations Sun. On peut les télécharger sur leur page : http://java.sun.com/products. Une VM pour Linux peut etre charger sur le site Java-Linux http://www.blackdown.org. Des VM existent pour quasiment toutes les plateformes, pour plus de détails voire le site de Sun.

Pour installer le programme, dézipper les archives dans un répertoire (par exemple /usr/local/corewarrior ou c:\corewarrior), c'est tout ! Pour une mise à jour, dézipper l'archive dans le répertoire du programme déja installé. Pour désinstaller le programme, il suffit d'effacer le répertoire.
