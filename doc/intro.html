J'ai le plaisir de vous présenter ma dernière réalisation : un environnement de développement pour Core War. Il permet d'éditer les programmes Red Code et de les exécuter dans un moteur qui délivre en temps réel une représentation du core.

Si tout cela vous semble obscure, lire l'historique de Core War et la doc sur son langage de programmation (le Red Code).

Je tiens ici à remercier :

  • Nat Makarevitch pour m'avoir ouvert les yeux sur Linux (et ne pas s'etre moqué de moi quand je tournais encore sous Windows :).
  • Xavier Cazin pour avoir su me convaincre de passer à Emacs (malgré ma résistance acharnée).

Note

Ce programme est encore en version béta, alors merci de faire preuve d'indulgence. D'autre part, je suis intéressé par toute suggestion en vue d'améliorer ce programme, et surtout de le rendre plus compatible avec le Red Code. Je vais m'efforcer de rendre le code généré compatible avec les dernières versions de MARS.
