Forum

Cette page contient une Applet Java de chat permettant aux joueurs de SpiceWars de se rencontrer et de se fixer des rendez-vous pour des parties endiablées. Après la connexion de l'Applet, veuillez taper votre pseudo puis appuyer sur [Enter]. Pour obtenir de l'aide concernant les commandes du chat, taper .help.

Vous pouvez connaitre votre numéro IP (pour fixer rendez-vous avec d'autres joueurs) en tapant la commande .whois <votre pseudo>. Les joueurs invités devront taper cette IP dans le champ "Adresse" du panneau de configuration du jeu.

Veuillez vous procurer un navigateur compatible Java
