Bonjour

Merci d'essayer Core Warrior. Core Warrior est un environnement de développement pour Core War (éditeur, compilateur et machine virtuelle).

Installation

Dézipper l'archive dans un répertoire (par exemple /usr/local/corewarrior ou c:\corewarrior). Changer le répertoire d'installation dans les scripts editer, moteur et compiler ou les mêmes en .bat pour les Windoziens. Si vous utilisez JRE, remplacer java par jre dans ces scripts, lire les remarques dans ces scripts.

On peut télécharger une machine virtuelle sur le site de Sun (java.sun.com/products). Je recommande le JRE 1.1.7. On peut trouver une VM pour Linux sur le site Java-Linux (www.blackdown.org). Il existe des VM pour quasiment toutes les plateformes, voire sur le site de Sun.

Utilisation

Commencer par lancer l'éditeur en lacant le script editer ou Editer.bat. Le reste est intuitif (du moins je l'espère :), donc suivez votre instinct. Se rappeler que les sources ont l'extension .src et les binaires .bin. Ne pas charger les binaires dans l'éditeur et les sources dans le moteur !
